# LogCal Feature Roadmap

## Priority Features

### 1. Macros Tracking
Track protein, carbs, and fat alongside calories.
- Display macros breakdown for each meal
- Daily macros summary on dashboard
- Set macro goals (not just calories)
- Visual pie chart or bar showing macro distribution
- AI already returns this data - need to surface it in UI

### 2. Quick Log
One-tap re-logging of frequently eaten meals.
- "Recent Meals" section on log screen
- "Favorites" that users can save
- Tap to pre-fill, adjust quantity if needed
- Smart suggestions based on time of day (e.g., show breakfast items in morning)

### 3. Weight Tracking
Simple weight logging with progress visualization.
- Log weight with date
- Graph showing weight over time
- Correlate with calorie intake trends
- Goal weight setting
- Weekly/monthly weight change summary

### 4. Apple Health Integration
Sync data with Apple Health app.
- Export calories to Health
- Export weight to Health
- Import weight from Health (if logged elsewhere)
- Import active calories/exercise data
- Privacy-conscious - user controls what syncs

### 5. Watch App
Quick logging from Apple Watch.
- View today's calories and remaining
- Voice logging ("2 eggs and toast")
- Complication showing daily progress
- Quick actions for common meals
- Sync with phone app

### 6. Widgets
Home screen widgets for quick access.
- Small: Today's calories / goal
- Medium: Today's calories + recent meals
- Large: Weekly chart preview
- Lock screen widget (iOS 16+)
- Interactive widget to quick-log (iOS 17+)

### 7. Siri Shortcuts
Voice-powered logging.
- "Hey Siri, log my lunch"
- "Hey Siri, how many calories today?"
- Custom shortcuts for frequent meals
- Shortcuts app integration

### 8. AI Coach
Chat-based nutrition assistant.
- Ask nutrition questions ("Is 2000 calories enough for me?")
- Get personalized tips based on eating patterns
- Goal adjustment suggestions based on progress
- Conversational interface within the app

### 9. Weekly AI Summary
AI-generated insights on your eating habits.
- Weekly report analyzing patterns and trends
- Identify habits (e.g., "You tend to exceed your goal on weekends")
- Personalized suggestions for improvement
- Anomaly alerts ("You logged 500 calories less than usual")
- Push notification with summary
